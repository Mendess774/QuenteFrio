# QuenteFrio

Advinhe qual número aleatório de 1 a 100 foi escolhido.

Conforme as tentativas, o programa ajudará a saber se o número escolhido está perto ou longe de ser acertado.

## _Exemplo

```
---> QUENTE E FRIO <---

Pressione uma tecla para continuar...

```
